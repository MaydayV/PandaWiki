# 开源版与乘风版功能对比

> 说明：本对比以当前仓库代码为准，用于帮助使用者快速了解“原版开源能力”与“乘风版增强点”的差异。

## 1. 产品定位

| 维度 | 原版开源版 | 乘风版 |
| --- | --- | --- |
| 产品名称 | PandaWiki 开源版 | Fly Version（乘风版） |
| 版本标识 | 原版语义版本 | `FV{大版本}.{功能序号}.{提交序号}.{原版版本号去点}` |
| 维护边界 | 原版社区与官方仓库 | 乘风版独立仓库与维护节奏 |

## 2. 管理后台与权限能力

| 功能项 | 原版开源版 | 乘风版 |
| --- | --- | --- |
| 后台版本展示 | 开源版标识 | 乘风版标识与介绍文案 |
| 专业版/商业版遮罩 | 部分功能有版本提示 | 已移除关键页面的版本遮罩 |
| MCP 设置入口 | 存在版本能力限制提示 | 已开放后台设置入口（不再显示商业版可用遮罩） |
| 统计设置 `pv_enable` | 依赖页面配置 | 默认开启并可配置 |
| 管理入口链接 | 原版仓库为主 | 乘风版仓库为主，原版仓库用于致谢与溯源 |

## 3. 核心业务流程

| 功能项 | 原版开源版 | 乘风版 |
| --- | --- | --- |
| 文档贡献流程 | 基础能力 | 已补齐可用后端流程（`list/detail/audit` 落库与状态流转） |
| 问答连续提问间隔 | 存在“敬请期待/未完整”场景 | 已补齐可配置与运行时拦截逻辑 |
| 内容复制尾巴 | 固定/有限能力 | 支持“增加内容尾巴”自定义内容，默认值可编辑 |
| 文档元信息展示 | 固定展示项 | 支持创建时间/更新时间/字数开关控制 |
| 文档属性访问稳定性 | 存在接口异常场景 | 已修复主要网络异常问题 |

## 4. 前台体验与国际化

| 功能项 | 原版开源版 | 乘风版 |
| --- | --- | --- |
| 中英文界面一致性 | 存在部分中文残留 | 已修复主要中英混显与文案空格问题 |
| 文档头部元信息排版 | 英文场景存在拼接问题 | 已优化为正确英文展示格式 |
| 前台 Logo 配置 | 固定资源 | 支持上传自定义 Logo（含 1:1 比例校验） |
| 欢迎页配色方案 | 存在设置不生效场景 | 已修复配置生效链路 |
| 统计地图 | 中国地图为主 | 已支持世界地图展示与数据拆分逻辑 |

## 5. 版本与运维策略

| 功能项 | 原版开源版 | 乘风版 |
| --- | --- | --- |
| 在线更新检测 | 原版流程 | 乘风版不依赖原版更新检测 |
| 授权激活/解绑流程 | 原版体系 | 乘风版不依赖该体系 |
| 数据库部署 SQL | 历史增量迁移链路 | 统一完整部署 SQL（fresh deploy） |
| 部署文档 | 原版为主 | 增加“手动安装 + 方案 B 镜像部署”文档 |

## 6. 使用建议

1. 生产部署优先使用镜像化方案（方案 B），便于升级与回滚。
2. 反馈问题时优先提交到乘风版仓库；仅在原版可复现的问题再去原版仓库反馈。
3. 对外提供服务时，请同步提供当前运行版本对应源码链接（AGPL 要求）。
